# PWAアイコン生成について

現在、画像生成のクォータに達したため、アイコンを自動生成できませんでした。

## アイコンの準備方法

以下の2つの画像ファイルを `public/` フォルダに配置してください：

### 必要なファイル

1. **icon-192.png** (192x192ピクセル)
2. **icon-512.png** (512x512ピクセル)

### アイコンのデザイン案

- **背景**: 紫-青のグラデーション (#667eea から #764ba2)
- **アイコン**: スマートフォン📱と時計⏰のシンボル
- **スタイル**: モダンでミニマル、ビジネス向け

### 簡易的な対応

アイコンが用意できるまで、以下の対応も可能です：

1. **Favicon Generator**を使用
   - https://favicon.io/favicon-generator/
   - テキスト「残」や「報」を入力してアイコン生成

2. **エモジをPNGで保存**
   - 📱 のエモジスクリーンショットを保存
   - 背景をグラデーションに加工

3. **一時的にスキップ**
   - アイコンなしでもアプリは動作します
   - PWAインストール時にデフォルトアイコンが使用されます

## 配置場所

```
public/
  ├── icon-192.png
  └── icon-512.png
```

アイコンを配置したら、PWAが正しく動作します。
