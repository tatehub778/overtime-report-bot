# トラブルシューティング: 時間入力ができない

## 確認事項

### 1. ブラウザのキャッシュクリア

**スーパーリロードを実行**:
- Windows: `Ctrl + Shift + R` または `Ctrl + F5`

### 2. デプロイ状況の確認

Vercelのデプロイが完了しているか確認:
1. https://vercel.com/dashboard にアクセス
2. `overtime-report-bot` プロジェクトを開く
3. 最新のデプロイが「Ready」になっているか確認

### 3. 開発者ツールで確認

1. `F12` キーで開発者ツールを開く
2. `Console` タブでエラーがないか確認
3. `Elements` タブで input 要素を確認:
   - `<input type="number" class="hours-input" ...>` が存在するか
   - `disabled` 属性がついていないか
   - スタイルで `display: none` や `visibility: hidden` になっていないか

### 4. ネットワークタブで確認

1. 開発者ツールの `Network` タブを開く
2. ページをリロード
3. `app.js` が正しく読み込まれているか確認（200ステータス）
4. ファイルサイズが約10KBあるか確認

## 一時的な対処法

もしデプロイが反映されていない場合:

1. 数分待ってからリロード
2. プライベートブラウジングモードで開く
3. 別のブラウザで試す

## 問題が続く場合

以下のスクリーンショットを共有してください:
- 開発者ツールのConsoleタブ
- 開発者ツールのElementsタブ（input要素の部分）
